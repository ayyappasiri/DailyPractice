## Learning Goal
Change column value with a pattern

### Problem statement
You need to pad zeros to the values from a column with number datatype.

Example:
- Change 1 to 0001
- Change 2 to 0002
- Change 3 to 0003
- Change 10 to 0010
- Change 100 to 0100
- Change 1000 to 1000

Maximum number of digits in the output  = 4
